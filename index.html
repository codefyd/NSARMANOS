بوابات موحدة للتسجيل، بيانات الطالب، وإدارة العاملين

واجهة خارجية لأولياء الأمور والطلاب، وواجهة داخلية للعاملين والإدارة، مع ربط مباشر عبر Google Apps Script وGoogle Sheets واستضافة الواجهة على GitHub Pages.

بوابة التسجيل والقبول بوابة بيانات الطالب بوابة العاملين
بعد ضبط config.js وربط رابط Google Apps Script المنشور، يمكنك استخدام زر التهيئة مرة واحدة فقط لإنشاء الشيتات والبيانات الأساسية.

بوابة التسجيل

استقبال طلبات الطلاب الجدد ومتابعتها حتى القبول أو الانتظار أو الرفض.

بوابة بيانات الطالب

استعراض بيانات الطالب، تعبئة الحقول الفارغة، ورفع طلبات التعديل عند الحاجة.

لوحة العاملين

لوحة موحدة للمعلم والمشرف والمدير لإدارة الطلاب والطلبات والإنذارات والإعدادات.
بوابات واضحة
فصل بين واجهة التسجيل، وواجهة بيانات الطالب، وواجهة العاملين لتجربة أسهل وأكثر ترتيبًا.
متابعة وإنذارات
دعم الإنذارات التعليمية والإدارية، مع إمكانية التحديث والمتابعة داخل لوحة العاملين.
ربط مباشر
الواجهة تعمل مباشرة مع Google Apps Script وGoogle Sheets بدون استضافة خادم تقليدي.
